SONATA (Scalable Open Network Architecture for Tabulated Annotations) is another standardized format, specifically designed for large-scale neural network simulations. It focuses on efficiency and scalability, making it suitable for simulating networks with millions of neurons. SONATA uses a combination of HDF5 and JSON files to store network structure, connectivity, and simulation parameters. This format is particularly useful for high-performance computing environments. Below is an example of a SONATA network configuration in JSON:

{
  "manifest": {
    "$NETWORK_DIR": "./network"
  },
  "target_simulator": "NEURON",
  "nodes": [
    {
      "nodes_file": "$NETWORK_DIR/nodes.h5",
      "node_types_file": "$NETWORK_DIR/node_types.csv"
    }
  ],
  "edges": [
    {
      "edges_file": "$NETWORK_DIR/edges.h5",
      "edge_types_file": "$NETWORK_DIR/edge_types.csv"
    }
  ]
}
